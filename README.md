# Simple Calculator using Tkinter

A simple and interactive **Calculator** application built using Python's built-in **Tkinter** library for the Graphical User Interface (GUI). This project is lightweight, responsive, and perfect for beginners practicing Python GUI development.

---

## ✨ Features

- **User-Friendly Interface:** Clean and classic button layout.
- **Basic Arithmetic Operations:** 
  - Addition (`+`)
  - Subtraction (`-`)
  - Multiplication (`×`)
  - Division (`÷`)
- **Clear Button (`C`):** To reset the expression.
- **Error Handling:** Displays error or handles division by zero gracefully
